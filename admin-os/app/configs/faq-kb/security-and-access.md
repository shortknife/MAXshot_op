# Security and Access FAQ

## How do I enable two-factor authentication?
Open the account security area, choose **Two-Factor Authentication**, and follow the authenticator setup flow. Accounts without two-factor enabled may be asked for additional verification during high-risk sign-ins.

## What happens after too many failed login attempts?
Repeated failed attempts temporarily lock the session challenge. The user should wait for the cooldown window or use the password reset path if access is still blocked.
