# 03.2 Execution 生命周期

> **用途**：Execution 状态流转定义

---

## 一、Execution 是什么

**Execution = 执行记录**，不可变历史。

---

## 二、生命周期

```
created
  → pending_confirmation（若 side_effect，可选）
  → confirmed / rejected / expired
  → executing
  → completed / failed
```

---

## 三、状态说明

| 状态 | 含义 |
|------|------|
| **created** | Orchestration 已创建 |
| **pending_confirmation** | 等待用户确认（side_effect） |
| **confirmed** | 用户确认，触发 Router |
| **rejected** | 用户拒绝 |
| **expired** | 确认超时（如 45 分钟） |
| **executing** | Router 执行中 |
| **completed** | 执行成功 |
| **failed** | 执行失败 |
