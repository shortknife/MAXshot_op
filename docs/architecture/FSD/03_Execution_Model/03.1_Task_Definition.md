# 03.1 Task 定义

> **用途**：Task 的元数据与可变性说明

---

## 一、Task 是什么

**Task = 任务定义**，可变配置。

- `task_id`
- `task_type`：ad_hoc / scheduled / long_running
- `schedule_config`（若 scheduled）
- `status`：active / paused / completed / failed

---

## 二、Task 类型

| 类型 | 含义 |
|------|------|
| **ad_hoc_task** | 一次性任务，创建后立即创建 Execution |
| **scheduled_task** | 定时任务，按 schedule 创建 Execution |
| **long_running_task** | 持续监控型，周期性创建 Execution |

---

## 三、Task 可变

Task 的配置、status 可随业务更新。  
Task 不是历史快照。
