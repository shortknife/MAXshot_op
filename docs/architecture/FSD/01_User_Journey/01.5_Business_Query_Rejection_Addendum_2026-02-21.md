# 01.5 Business Query Rejection Addendum (2026-02-21)

> 类型：Addendum（补丁文件）  
> 作用范围：补充 `01.1`、`01.2`、`01.4`，不覆盖原文

## 1. 新增拒绝类型（业务查询专用）

- `out_of_business_scope`：问题超出 MAXshot 业务域。
- `insufficient_business_data`：业务数据不足，无法给出可靠结论。
- `source_not_connected`：目标数据源当前不可达。
- `unsafe_write_attempt`：用户请求触发未授权写操作。

## 2. 输出规则

- 输出语言跟随用户输入语言。
- 对外文案优先“产品化短句 + 下一步建议”，内部日志保留技术码。

## 3. 文案模板（可本地化）

1. `out_of_business_scope`
- 面向用户：该问题不在当前业务查询范围内。请改问 Vault、执行日志或业务指标相关问题。

2. `insufficient_business_data`
- 面向用户：当前缺少足够业务数据，暂时无法得出可靠结论。请补充时间范围或对象。

3. `source_not_connected`
- 面向用户：数据源暂时不可用，请稍后重试。

4. `unsafe_write_attempt`
- 面向用户：该请求涉及写入操作，当前未授权执行。
