# 00.1 One-Pager：系统意图

> **用途**：30 秒内让技术同学理解 MAXshot 是什么、不是什么  
> **版本**：v5.1 执行版（继承 v5.0 宪法，务实收敛）

---

## 一句话

MAXshot 是一个 **小微团队增长操作系统（Small Team Growth OS）**，本质是**受控、可审计、可回放**的能力网络 + 进化引擎 + 灵魂滤镜，不是「智能聊天机器人」。

---

## 系统意图

| 维度 | 是什么 | 不是什么 |
|------|--------|----------|
| **本质** | 能力网络 + 进化引擎 + 灵魂滤镜 | 通用 AI 对话系统 |
| **决策** | 确定性调度（Router） + 显式 Policy | LLM 黑箱决策 |
| **记忆** | 原子化组装（Foundation/Experience/Insight） | 无限对话历史 |
| **进化** | v5.1：策略闭环（标签+归因+周期复盘）；v5.0 愿景：假设-验证-反思 | 自动调参、在线学习 |
| **灵魂** | 滤镜（Filter），无 Runtime API | 独立服务、可调用模块 |

---

## 三大红线（不可违反）

1. **Router = 确定性调度器**，LLM 只做建议，不做调度
2. **Execution = 不可变历史**，Snapshot 启动时冻结
3. **不留痕 = Bug**，所有决策必须可审计、可回放
