# 04.1 Working Mind 总览

> **用途**：理解「实时心智合成」是什么

---

## 一、定义

**Working Mind** = 针对当前 Task 实时合成的认知上下文。

---

## 二、合成公式

```
Working Mind = Soul (Filter) + Retrieved Memory Atoms + Current Goal
```

---

## 三、流程

1. **Filter**：透过 Soul 滤镜过滤选项
2. **Retrieve**：从 Memory 库召回 Foundation / Experience / Insight 原子
3. **Synthesize**：合成为 `context.memory_refs`，注入 Capability
